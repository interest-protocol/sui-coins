## Quick start

Install the project dependencies with yarn

```bash
  yarn
```

Initialize the application

```bash
  yarn dev
```

### Run tests

First of all, make sure you have `jest` installed, if not, follow the steps below.

```bash
  yarn add --dev jest
```

Only then will you be able to run the following command

```bash
  yarn test
```
